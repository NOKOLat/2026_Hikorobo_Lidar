# ビルドと実行方法

- このリポジトリではスクリプトを用いてビルドとシステム起動を行います。以下の手順に従ってください。

- 設定のymlファイルを変更してもビルドは不要です。システム起動時に自動で読み込まれます。

### ビルド

```bash
./build.sh
```

### システム起動

```bash
./start.sh
```

- 実行するとオプションを選択するメニューが表示されます。

- 実行したい機能の番号を入力してください。

**オプション:**
1. データ取得のみ (lidar_get_points)
2. 前処理ノード (lidar_preprocess)
3. 背景差分ノード (lidar_background_diff)
4. 全て同時起動 (データ取得 → 前処理 → 背景差分)
